I have set this up using the Live Server extention on VS Code, but you can also simply download the file and open it on your browser. It should create a tab where everything works =)

To clarify:

Method 1:
Download the file and open it with your preferred browser.

Method 2 (VS Code):
Make sure the file is in a folder on your workspace, and that you have the Live Server extension installed.

Click Go Live on the bottom right corner. That should open a tab on your browser, where you can start tinkering.
